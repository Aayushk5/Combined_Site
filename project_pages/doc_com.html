IP-address & HTTP

December 16th, 2020 by Aayush Kumar

Information is transmitted over the network in discrete chunks called packets; each packet is mostly made up of whatever data the sender is trying to communicate.

IP Address

IP address is one among the data stored. An IP address is a number used to label any device connected to a network on which the Internet Protocol is used as the medium for communication. IP stands for Internet Protocol

There are two types of IP addresses IPv4 and IPv6.

HTTP

HTTP stands for Hyper Text Transfer Protocol. It is an application layer protocol for the transmission of hypermedia documents. Communication between client computers and web servers is done by sending HTTP Requests and receiving HTTP Responses. A client establishes a connection with a server and sends a message with the request data. The server responds with a similar message, containing the status of the operation and its possible result.
